Server commands

E-mail of the server: wasteland-pbem@yandex.ru

How to join the game

Send email to server addres with following lines inside. Note that subject of the join request email must be "request" (exactly that one word), or else request will not be processed. This, and all other emails should be in plain text format, not html - server can't process html properly.

Faction: Faction Name
Chosen: man (or: woman)
Name: Name of Chosen One
Skill: one of: PYRO, STAI, SGUN*

* Skill parameter defines a special skill available to your Chosen. Only one skill might be taken. More skill may be added later, so keep an eye for updates if you'll want to restart.

  • pyromania [PYRO]: Addiction to various hand grenades.
  • specialist tailor [STAI]: Ability to produce special clothes.
  • specialist gunsmith [SGUN]: Ability to produce double-barrelled shotguns.

How to send an order

Send email to server with subject "orders" and your orders in message body, for example:

#orders 123 "password"

person 132
  produce RAT

#end

How to ask the server to re-send last turn report

#resend 123 "password"

How to send message to other player

For open discussion of game-related questions, subscribe to wasteland-radio. Primary language of the Radio is Russian, but English is allowed as well and will be understood by majority of players.

To send a message to faction number (for example, 155), send following email:

#mail 155 123 "password"

Hello there!

#end

Parameters 123 "password" is your faction number and password; you may omit these if you don't want the message to be signed.

Message to person number if faction is unknown (for example, 1536):

#mail person 1536 123 "password"

Technical questions

If you want to discuss game engine, look in the source code or report a bug, subscribe to wasteland-pbem list. Again, primary language is Russian. Look for sources in Files section.
