Список команд

  • команда для лидера: приказ нужно отдавать командиру бригады.
  • на полный ход: персонажу можно отдать только один полноходовой приказ в каждом ходу.

ADDRESS

ADDRESS "mail@mail.ru"

Сменить e-mail, на который отсылается ваш отчёт.

ATTACK

ATTACK 123

Команда для лидера.

Атаковать указанного персонажа.

AVOID

AVOID 1
AVOID 0

"Избегать боя". Персонаж с установленным флагом AVOID будет по возможности убегать из боя, если атакован. Если отряд находится в патруле, этот флаг автоматически снимается.

BUILD

BUILD "Tank"

Команда для лидера, на полный ход.

Начать строительство нового объекта указанного типа. Если постройка не ограничивается одним ходом, на следующем ходу надо отдать команду INSTALL.

BURN

BURN WOOD GAS

Установить порядок предпочтений персонажа при выборе топлива для обогрева в холода. В данном примере, персонаж вначале сожжёт всё имеющееся у него дерево, и только потом начнёт жечь бензин.

BURN

Восстановить порядок, заданный по умолчанию.

CONSUME

CONSUME RAT CAN

Установить порядок предпочтений персонажа при выборе еды. В данном примере, персонаж вначале съест всех имеющихся у него крыс, и только потом примется за банки консервов.

CONSUME

Восстановить порядок, заданный по умолчанию.

CURE

CURE 123 134 145

Команда для лидера, на полный ход.

Лечить указанных персонажей.

DECLARE

DECLARE 12 Neutral

Установить отношение к указанной фракции.

DECLARE DEFAULT Neutral

Установить отношение ко всем фракциям, для которых отношение не указано особо.

DESCRIBE

DESCRIBE PERSON "This is the person"
DESCRIBE OBJECT "This is the building"

Установить описание персонажа или объекта. Нужно быть владельцем объекта, чтобы сменить описание.

DRIVE

DRIVE [список направлений]
DRIVE ATTACK [список направлений]

Команда для лидера, на полный ход.

Ехать внутри объекта в указанном направлении. Если указано более чем одно направление, персонаж будет пытаться проехать так далеко, как сможет. Чтобы выполнить эту команду, персонаж должен находиться внутри транспорта, быть его владельцем (т.е. стоять первым в списке персонажей внутри) и уметь им управлять.

Используются следующие направления: N, NW, SW, S, SE, NE (пример: DRIVE N).

Если указана директива ATTACK, персонаж будет атаковать патрульных, которые попытаются препятствовать его продвижению.

ENTER

ENTER 123

Команда для лидера.

Войти в указанный объект. Если приказ отдается отряду, находящемуся в другом объекте, он должен сперва покинуть первый объект. Выполнение команды будет успешным только в том случае, если указанный объект не занят, или принадлежит фракции, относящейся к вам по-дружески.

EQUIPMENT

EQUIPMENT "kalashnikov" "makarov" "pipe rifle" "steel armor" "leather jacket"

Установить порядок предпочтений при выборе оружия, брони и взрывчатки.

EQUIPMENT

Восстановить порядок, заданный по умолчанию.

EVICT

EVICT 123

Выселить другого персонажа из объекта, владельцем которого является данный. Выселить персонажа в бригаде нельзя - только командира бригады или отдельного персонажа.

GIVE

GIVE [персонаж] [количество] [предмет]
GIVE 123 10 "rats"
GIVE 123 ALL "rats"

Передать нужное количество вещей другому персонажу.

Если персонаж принадлежит другой фракции, эта фракция должна относиться к вам как минимум дружески (friendly).

Если вместо номера персонажа стоит 0, вещь будет выброшена.

GREEDY

GREEDY 0
GREEDY 1

"Не делиться с другими персонажами". Персонаж, у которого установлен этот флаг, не будет автоматически делиться с другими едой и топливом. К приказу GIVE это не относится - все отданные вами команды передачи будут исполнены.

HIDE

HIDE NONE

Персонаж не должен скрывать свою фракцию.

HIDE FACTION

Персонаж должен скрывать свою фракцию.

INSTALL

INSTALL "car details" "engine"

Команда для лидера, на полный ход.

Вести строительство объекта, устанавливая в него необходимые компоненты. Персонаж должен находиться внутри объекта. Можно указывать несколько компонентов, но требуемый для их установки навык должен быть одинаковым для всех.

KICK

KICK

Выгнать персонажа из фракции. Избранного выгнать нельзя.

LEAVE

LEAVE

Команда для лидера.

Выйти из объекта, в котором персонаж находится в настоящий момент.

MOVE

MOVE [список направлений]
MOVE ATTACK [список направлений]

Команда для лидера, на полный ход.

Двигаться в указанном направлении. Если указано более чем одно направление, персонаж будет пытаться пройти так далеко, как сможет.

Используются следующие направления:

Стороны света - N, NW, SW, S, SE, NE (пример: MOVE N).
Номер объекта, в который надо войти (пример: MOVE 12).
OUT, если надо выйти (пример: MOVE OUT).

Если указана директива ATTACK, персонаж будет атаковать патрульных, которые попытаются препятствовать его продвижению.

NAME

NAME PERSON "название"
NAME FACTION "название"
NAME OBJECT "название"

Сменить соответствующее имя. Чтобы переименовать объект, персонаж должен быть его владельцем, т.е. стоять первым внутри него.

OPTION

OPTION xml-report [1 | 0]

Получать ежемесячный отчёт в виде xml. По умолчанию 0.

OPTION text-report [1 | 0]

Получать текстовый ежемесячный отчёт. По умолчанию 1.

OPTION language [ru | en]

Сменить язык отчёта.

OPTION template [long | short]

Показывать или скрыть дополнительные комментарии в шаблоне приказа. По умолчанию short.

PASSWORD

PASSWORD "новый пароль"

Сменить пароль фракции, который указывается в первой строке файла приказов (#orders ...).

Важное примечание: пароль поменяется только во время обработки следующего хода. Приказ, в котором дана эта команда, отсылается ещё под старым паролем. Если вы решили поменять приказы до обработки хода, их нужно отсылать также со старым паролем.

PATROL

PATROL

Команда для лидера, на полный ход.

Патрулировать регион. Патрульные будут по возможности принимать на себя все атаки, направленные на персонажей вашей фракции или союзных (ally) фракций. Кроме того, они будут блокировать недружественным (unfriendly) персонажам вход в охраняемый регион.

PRODUCE

PRODUCE "rats"

Команда для лидера, на полный ход.

Весь месяц заниматься производством указанного предмета.

QUIT

QUIT "password"

Выйти из игры. При отдаче такой команды ваш Избранный совершает самоубийство, а все персонажи фракции распускаются. В качестве параметра требуется дополнительно указать пароль; это сделано для обеспечения некоторой защиты от случайно выданного приказа.

SCAVENGE

SCAVENGE
SCAVENGE "cans" "metal constructions" ...

Команда для лидера,на полный ход.

Подобрать некоторое количество предметов из Мусора. За каждый уровень навыка "мусорщик" персонаж может подобрать один любой предмет. Можно указывать несколько разных типов предметов. Если ничего не указано, персонаж будет пытаться подобрать всё, что лежит в регионе.

SHOW

SHOW SKILL "scavenge"
SHOW ITEM "rat"
SHOW OBJECT "Tank"

Включить в отчёт подробное описание указанного предмета, навыка или объекта.

SPOILS

SPOILS "dead rats" "meat"

Указать предпочтения персонажа при сборе трофеев. Можно указывать несколько типов предметов, но на самом деле подбираться будут предметы только одного из этих типов.

SPOILS

Разрешить персонажу самому выбирать трофеи.

TEAM

TEAM 123

Поступить под начало заданного персонажа. Если указанный персонаж относится к другой фракции, она должна относиться к вам дружественно (friendly). Если персонаж, которому отдана команда, сам является лидером, его бригада также переходит к новому лидеру. Если новый лидер сам является подчинённым, он автоматически выходит из своей бригады.

TEAM (или TEAM NONE)

Выйти из бригады. Если команда отдана лидеру, его бригада расформировывается.

TEAM KICK 123

Команда для лидера.

Выгнать из бригады заданного персонажа.

TRADE

TRADE 5 "rats" 1 "ration"

Предложить 5 крыс в обмен на 1 рацион. Предложение действует для любого персонажа в регионе.

TRADE WITH 123 5 "rats" 1 "ration"

Предложить такую же сделку конкретному персонажу.

Этот приказ не является приказом на полный ход, но персонаж может выполнять только один приказ TRADE.

UNINSTALL

Команда для лидера, на полный ход.

Вытаскивать из объекта указанные компоненты. Персонаж должен находиться внутри объекта; владельцем объекта должен быть персонаж той же фракции.

UNINSTALL "car details" "engine"

В этой форме приказа персонаж будет пытаться демонтировать максимальное количество указанных компонентов. Можно указывать несколько компонентов, но навык, требуемый для демонтажа, должен быть одинаков.

UNINSTALL 10 "car details"

В этой форме можно указать только один тип компонента и задать количество, которое должно быть демонтировано.

Последовательность действий

Приказы выполняются в следующем порядке:

  • Настроечные приказы: ADDRESS, AVOID, EQUIPMENT, BURN, CONSUME, DESCRIBE, GREEDY, NAME, OPTION, PASSWORD,QUIT, SHOW, SPOILS
  • ATTACK (и автоматические атаки на враждебных персонажей)
  • Быстрые приказы: DECLARE, TEAM, LEAVE, ENTER, EVICT, HIDE, KICK
  • TRADE, GIVE
  • Полномесячные приказы: BUILD, CURE, INSTALL, PATROL, MOVE, DRIVE, PRODUCE, SCAVENGE, UNINSTALL
  • Персонажи едят
  • Обрабатывается климат и радиация
  • Рождаются дети и делаются новые
  • Сумасшедшие выходят из фракции
